


## Image Requirements

### Hero Image (`hero/hero1.jpg`)
- **Dimensions:** 1920x1080px or larger
- **Format:** JPG
- **Usage:** Main hero section background on homepage
- **Theme:** Modern real estate, houses, or architectural exteriors

### Property Images (`properties/p1.jpg`, `p2.jpg`, `p3.jpg`)
- **Dimensions:** 800x600px or larger
- **Format:** JPG
- **Usage:** Property cards in listings and featured sections
- **Theme:** Different property types (villa, apartment, house)

### About Image (`about/about.jpg`)
- **Dimensions:** 1200x800px or larger
- **Format:** JPG
- **Usage:** About page content section
- **Theme:** Real estate team, office, or professional real estate setting

### Logo (`logo.svg`)
- **Format:** SVG (scalable vector)
- **Usage:** Splash screen, header (if needed)
- **Colors:** Primary green (#2ecc71) background with white text

## Note

If you need to replace any images:
